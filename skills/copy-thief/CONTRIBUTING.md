# Contribuindo

Obrigado pelo interesse. Este projeto é uma skill de copywriting (PT-BR), feita de Markdown e um
HTML estático. Não há build, dependências ou testes.

## Como contribuir

- **Bug ou erro de conteúdo:** abra uma issue descrevendo o que está errado e onde (arquivo e
  trecho).
- **Sugestão de melhoria:** abra uma issue explicando o problema que ela resolve antes de mandar um
  PR grande.
- **Pull request:** faça um fork, crie um branch a partir de `main`, faça a mudança e abra o PR
  apontando o porquê.

## Regras de estilo (valem para todo o conteúdo)

1. Idioma: PT-BR, mercado brasileiro.
2. Nunca use travessão `—` nem hífen isolado ` - ` como pontuação. Troque por vírgula, ponto, dois
   pontos ou parênteses. Hífen dentro de palavra é permitido (e-mail, no-code, passo-a-passo).
3. Não fabrique prova (depoimentos, números, resultados). Use placeholders explícitos.
4. Mantenha os arquivos curtos e com um assunto só. Detalhe operacional vive em `SKILL.md`; o mapa
   do repo está em `AGENTS.md`.

## O que não enviar

Não adicione conteúdo raspado de terceiros ao repositório (a pasta `.firecrawl/` é ignorada de
propósito). Contribua com síntese própria, não com cópia verbatim de artigos protegidos.
