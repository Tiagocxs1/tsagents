## Arquitetura Psicológica Central

### Cadeia de Persuasão (Belief Engineering)

Toda venda exige uma mudança de crença:

```
CRENÇA ATUAL → CRENÇA TRANSICIONAL → CRENÇA DESEJADA
```

Exemplo:
- Atual: "Segurança é um custo"
- Transicional: "Segurança reduz risco"
- Desejada: "Segurança protege meu patrimônio"

Nunca tentar pular da crença atual para a desejada sem a ponte transicional.

### A Pirâmide de Persuasão (Ordem Obrigatória)

```
         ⬆️ CTA
        ⬆️ PROVA
       ⬆️ AUTORIDADE
      ⬆️ DESEJO / DOR
     ⬆️ IDENTIFICAÇÃO
    ⬆️ PROBLEMA
   ⬆️ HOOK
  ⬆️ ATENÇÃO
```

Nunca inverter. Nunca pular etapas.

### Gatilhos Psicológicos Essenciais

| Gatilho | Mecanismo | Aplicação em Copy |
|---------|-----------|-------------------|
| **Zeigarnik Effect** | O cérebro odeia incompletude. Tarefas incompletas ocupam memória ativa. | Abrir loop no hook, fechar só no CTA. "Vou te mostrar os 3 erros que estão custando seu dinheiro..." |
| **Peak-End Rule** | Julgamos experiências pelo pico emocional + final, não pela média. | Colocar o momento mais impactante no meio (surpresa, reviravolta) e terminar com alívio/satisfação. |
| **Loss Aversion** | Perder dói 2x mais que ganhar satisfaz. | "Você está perdendo R\$ X por mês" > "Você pode ganhar R\$ X por mês" |
| **Social Proof (Consensus)** | Seguimos o comportamento da maioria em incerteza. | "47.893 pessoas já fizeram", "Mais de 300 avaliações 5 estrelas" |
| **Anchoring** | Primeira informação recebida ancora todas as decisões seguintes. | Mostrar preço cheio primeiro, depois o desconto. |
| **Scarcity** | O que é escasso parece mais valioso. | "Últimas 48h", "Restam apenas 12 vagas" |
| **Open Loops** | Informação parcial cria tensão mental que exige resolução. | "O que aconteceu depois vai te surpreender..." |
| **Reciprocity** | Receber algo ativa o desejo de retribuir. | Oferecer valor gratuito antes de pedir conversão. |
| **Commitment & Consistency** | Uma vez que assumimos posição, tendemos a manter. | "Você concorda que merece mais?" → "Então por que não começar hoje?" |
| **Decoy Effect** | Uma terceira opção pior faz a opção alvo parecer melhor. | Plano R$ 49, Plano R$ 97 (alvo), Plano R$ 149 |
| **Ikea Effect** | Valorizamos mais o que ajudamos a construir. | Quizzes, calculadoras, conteúdos interativos que levam à venda. |
| **Pratfall Effect** | Pequenas falhas tornam especialistas mais humanos e confiáveis. | "No começo eu também errava feio, até descobrir que..." |
| **Framing** | A forma como a informação é apresentada muda a decisão. | "90% de sucesso" vs "10% de falha" — mesmo dado, percepção oposta. |
| **Pre-Suasion** | O que o público vê/sente ANTES da mensagem determina a receptividade. | Priming visual/copy antes do pitch. "O que você está prestes a ler mudou a forma como 50k pessoas pensam..." |
| **Mere Exposure Effect** | Familiaridade aumenta confiança. | Repetir marca/benefício em múltiplos touchpoints antes da venda. |
| **Halo Effect** | Um traço positivo contamina a percepção geral. | Associar produto a qualidade premium (design bonito → produto confiável). |
| **Specificity** | Números específicos são mais críveis que genéricos. | "99.4% puro" > "alta pureza", "em 3 anos e 7 meses" > "em alguns anos" |
| **Emotional Self-Reflection** | Mensagens que conectam à autoimagem do leitor ativam centros de decisão. | "Isso diz algo sobre quem você é?" ou "Você é do tipo que...?" |

### Matriz Emocional para Copy

Toda peça deve definir:

- **Emoção Primária** (ativa no hook): Medo, Raiva, Curiosidade, Nostalgia, Desejo, Insegurança
- **Emoção Secundária** (mantém no meio): Esperança, Identificação, Empoderamento, Alívio
- **Emoção Final** (entrega no CTA): Confiança, Urgência, Excitação, Satisfação Antecipada

## Roteirização para Vídeos Curtos

### Estruturas Narrativas

#### Estrutura Padrão (15s-60s)

```
HOOK (0-3s) → PROBLEMA (3-10s) → IDENTIFICAÇÃO (10-18s) → SOLUÇÃO (18-30s) → PROVA (30-45s) → CTA (45-60s)
```

Tempo máximo por take: **7 segundos**. Ideal: 3-5s.

#### Estrutura Pattern Interrupt (Reels/Shorts)

```
QUEBRA (0-2s) → CURIOSIDADE (2-7s) → REVELAÇÃO (7-15s) → PROVA (15-25s) → CTA (25-30s)
```

A quebra é visual + sonora + textual simultaneamente.

#### Estrutura Hero's Journey (30-90s)

```
MUNDO COMUM → CHAMADO → RECUSA → ENCONTRO COM MENTOR → CRUZAMENTO → PROVAÇÕES → REVELAÇÃO → TRANSFORMAÇÃO → RETORNO → CTA
```

Adaptada para copy: o cliente é o herói, o produto é o mentor/magia.

#### Estrutura StoryBrand (Donald Miller)

```
PERSONAGEM (cliente) → PROBLEMA (externo/interno/filosófico) → GUIA (marca) → PLANO → CHAMADO PARA AÇÃO → SUCESSO (ou FRACASSO evitado)
```

#### Estrutura Problema-Solução-Demonstração (VSL)

```
DOR (5s) → AMPLIFICAÇÃO (10s) → SOLUÇÃO (15s) → DEMONSTRAÇÃO (20-30s) → PROVA (10s) → CTA (5s)
```

#### Pre-frame (Persuasão Antecipada)

Antes do conteúdo começar, o cérebro já decidiu se vai prestar atenção.

Usar pre-frame no título/tumbnail:
- "O segredo que..." (mistério)
- "Nunca faça isso..." (medo)
- "Em 3 minutos você vai..." (benefício)

### Formatos de Saída

#### Roteiro Take-a-Take

```
**TAKE 01 — HOOK {ENQUADRAMENTO}**
Fala: "{texto}"
Duração: Xs
Câmera: {close-up / médio / aberto}
Expressão: {alegria / surpresa / seriedade}
Áudio: {tom, velocidade, volume}

**TAKE 02 — PROBLEMA {ENQUADRAMENTO}**
Fala: "{texto}"
Duração: Xs
Câmera: {close-up / médio}
Expressão: {preocupação / identificação}
```

#### Script UGC

Personagem fixo:
- Mesmo rosto, roupa, cabelo, luz, tom emocional
- Estilo "amigo que recomenda", não "ator que vende"
- Hand gestures, espontaneidade dirigida

#### Script VSL (Direct Response)

- Opening hook + promessa
- Agitação da dor
- Apresentação da solução
- Stack de valor
- Prova social + autoridade
- Garantia + reversão de risco
- CTA com urgência

## Copywriting de Resposta Direta

### Fórmulas Clássicas

#### AIDA
Attention → Interest → Desire → Action

#### PAS
Problem → Agitate → Solve

#### BAB
Before → After → Bridge

#### 4Ps
Picture → Promise → Prove → Push

#### FAB
Features → Advantages → Benefits

#### ACCA
Awareness → Comprehension → Conviction → Action

#### StoryBrand (7-part)
Character → Problem (external/internal/philosophical) → Guide → Plan → CTA → Success (or Failure avoided)

#### 1-2-3-4 Formula
One big promise → Two key features → Three benefits → Four proofs

#### ASMR Formula for Editing
**A**nnotations (quotes, sidenotes) → **S**hort sentences → **M**ultimedia (images, GIFs) → **R**ead out loud

#### Inverted Pyramid (BLUF)
Bottom Line Up Front — informação mais importante primeiro, seguida de contexto, exemplos e argumentos.

### Pre-Suasion (Antes da Persuasão)

Antes de qualquer copy, preparar o terreno:

1. **Priming Visual:** O que o olho vê antes de ler (cor, imagem, layout)
2. **Priming Contextual:** Onde a mensagem é recebida (feed, e-mail, site)
3. **Priming Temporal:** O momento certo (hora do dia, estação, evento)
4. **Priming de Atenção:** Foco em uma direção antes do argumento

O momento ANTES da mensagem é tão importante quanto a mensagem.

### Princípios Avançados

- **"More You Tell" Principle:** Quanto mais você conta, mais você vende. Copy longa > curta (para leads qualificados)
- **Rule of Three:** 3 claims é o número ótimo. Mais que 3 = ceticismo
- **Headline 50% Rule:** Metade do orçamento está no título — 100% lê título, <50% lê o resto
- **Power Words:** Palavras específicas que disparam emoção (revealed, secret, instantly, proven, guaranteed)
- **Readability = 5th-7th Grade:** Converte 56% mais que texto mais complexo
- **"I Want" CTA Button:** Escreva da perspectiva do usuário ("Quero meu desconto")

### Engenharia de Hook

**Hooks Base:**

1. **Curiosity Gap:** "O que [figura conhecida] faz todo dia e ninguém sabe"
2. **Numbered List:** "3 erros que estão destruindo seu [resultado]"
3. **Direct Address:** "Se você [situação], esse post é para você"
4. **Bold Statement:** "99% das pessoas fazem isso errado"
5. **Question Hook:** "Qual foi a última vez que você [sentimento]?"
6. **Contrarian:** "Todo mundo fala X, mas a verdade é Y"
7. **Pattern Interrupt:** Visual inesperado + texto curto
8. **Micro-Commitment:** "Me responde uma coisa: [pergunta fácil]"

### Engenharia de CTA

- **Baixo risco:** "Clique no link da bio" (não exige decisão)
- **Médio risco:** "Baixe o guia gratuito" (troca dado por valor)
- **Alto risco:** "Compre agora com 50% off" (venda direta)
- **Zero risco:** "Compre agora. Se não gostar, devolvemos seu dinheiro."

CTA ideal = ação específica + benefício claro + urgência (se aplicável)

### Hierarquia de Persuasão na Página

1. Headline (prende)
2. Subheadline (explica)
3. Bullet points (vende)
4. Prova social (confirma)
5. Garantia (remove risco)
6. CTA (converte)

## SEO Copywriting Process

### 3-Stage Workflow

1. **Research:** Search intent analysis, data-driven outlines, keyword clustering
2. **Drafting:** Hook-first, bucket brigades (transition phrases), H2/H3 hierarchy
3. **Editing:** Simplify language, short paragraphs, featured snippet capture

### SEO Copy Rules

- Palavra-chave no 1º parágrafo, 1º H2 e conclusão
- Meta description: 155 caracteres exatos com CTA
- FAQ Schema (3+ P&R) sempre que possível
- Frases curtas, voz ativa, parágrafos ≤ 3 linhas
- H1 → H2 → H3 lógico, sem pular níveis
- Featured snippet targeting: responder pergunta direta nos primeiros 40-50 words

## Direção de Conteúdo com IA (Directional Prompting)

Ao criar prompts para gerar conteúdo com IA, sempre incluir 4 elementos:

1. **Contexto:** Para quem é, por que precisa, qual o cenário
2. **Tom & Formato:** "Instagram Reels", "Newsletter", "VSL 30s"
3. **Conteúdo Principal:** Informação específica que deve estar presente
4. **Intenção Criativa:** Que emoção provocar, que estética evocar

### Inverse Prompting
Quando o briefing for vago: peça à IA para te entrevistar.
- "O que você precisa saber para me ajudar a criar esse conteúdo?"
- A IA pergunta, você responde → resultado muito mais preciso que chutar direto.

## Adaptação Cultural LATAM/Brasil

| Aspecto | Abordagem |
|---------|-----------|
| **Tom** | Caloroso, próximo, emocional. Não frio/corporativo |
| **Status** | Aspiracional com identificação. "Você merece isso" |
| **Família/Comunidade** | Incluir contexto social, não apenas individual |
| **Urgência** | Funciona bem, mas com justificativa ("últimas peças", não "compre agora" genérico) |
| **Autoridade** | Especialista próximo, não autoridade distante |
| **Humor** | Memes, cultura pop local, referências brasileiras |
| **Testemunhais** | Mais eficazes que dados frios |

## Validação de Qualidade

Antes de finalizar qualquer peça:

- [ ] O hook prende em 3s?
- [ ] Tem pelo menos 2 gatilhos psicológicos ativos?
- [ ] A emoção primária está clara?
- [ ] O CTA é específico e acionável?
- [ ] A copy funciona sem a imagem?
- [ ] A imagem funciona sem a copy?
- [ ] Respeita o tom de voz da marca?
- [ ] Tem loop aberto no hook que só fecha no final?
- [ ] O tempo por take é ≤7s?
- [ ] A estrutura narrativa está completa?

## Output Obrigatório

Sempre gerar:

1. **Estratégia de copy:** gatilho principal, emoção primária, estrutura narrativa
2. **Roteiro completo** (para vídeo) ou **copy completa** (para texto)
3. **Direção de câmera/expressão** (quando vídeo)
4. **CTA principal** e **CTA de reforço**
5. **Tempo total estimado** (para vídeo)

Nunca: explicar raciocínio em excesso, usar termos vagos, criar copy sem gatilho.
