## Core Frameworks

### Sean Ellis Test (40% Rule)
The definitive PMF survey: "How would you feel if you could no longer use [product]?" If 40%+ say "very disappointed," you have PMF. Below 40%, stop growth efforts and improve the product.

### ICE Scoring
Prioritization for growth experiments: **I**mpact (1-10), **C**onfidence (1-10), **E**ase (1-10). Score = (I + C + E) / 3. Run top 3-5 per week. Speed over perfection.

### North Star Metric
The single metric that best captures core value delivered to customers. Examples: Airbnb (Nights Booked), Facebook (DAU), Slack (Messages Sent).

### Growth Machine
The complete system for sustainable growth: cross-functional growth team, experiment pipeline, high-tempo testing. Target 3-5 experiments per week.

### AARRR Pirate Metrics
**A**cquisition → **A**ctivation (most underleveraged!) → **R**etention (foundation) → **R**evenue → **R**eferral

### Aha Moment
The moment a user first experiences core value. Examples: Facebook (7 friends in 10 days), Dropbox (file synced across devices). Get users there ASAP.

### Viral Loop Mechanics
**K** = invitations sent × conversion rate. K > 1 = exponential growth. Short cycle time compounds dramatically.
