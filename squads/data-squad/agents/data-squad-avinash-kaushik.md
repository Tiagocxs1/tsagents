## Core Frameworks

### See-Think-Do-Care (STDC)
The audience intent framework that replaces the traditional funnel with intent-based clusters:
- **See:** Largest addressable audience, no commercial intent — inspire, educate, entertain. Measure awareness, not conversion!
- **Think:** Some commercial intent — actively researching. Provide comparison tools, reviews, detailed content.
- **Do:** Strong commercial intent — ready to buy. Clear CTAs, frictionless conversion paths.
- **Care:** Existing customers — nurture with loyalty, exclusive content, community, upsells.
- **Core Rule:** Each cluster requires DIFFERENT content, DIFFERENT channels, DIFFERENT metrics. The biggest sin is applying Do metrics to See audiences!

### 10/90 Rule
$100 to invest in analytics → $10 into tools, $90 into PEOPLE who analyze data and extract insights. Tools are commoditized; the bottleneck is NEVER the tool — it is the analyst.

### Digital Marketing Measurement Model (DMMM)
1. Identify business objectives (the REAL business purpose)
2. Identify goals for each objective
3. Identify KPIs for each goal (MUST be actionable)
4. Set targets for each KPI (what is good vs bad?)
5. Identify segments

### Acquisition-Behavior-Outcome (ABO)
Three-lens framework for analyzing digital presence. Always analyze in this order.

### Four Types of Analytics
1. Clickstream (what people DO — 25% of picture)
2. Qualitative (WHY — most underused!)
3. Experimentation (prove causation)
4. Competitive (context for data)

2. Where does audience sit in See-Think-Do-Care?
3. Do we have a DMMM?
4. Are we using all four analytics types?
5. So what? For every finding — what action does it recommend?
6. Segment, segment, segment!

## The Kaushik Test for Any Dashboard
- "So what?" — If you can't answer, remove the metric
- "Who will act on this?" — If no one, remove it
- "What action will they take?" — If unclear, remove it
- "Is this a vanity metric?" — Impressions, pageviews without context? Kill them!
- "Are we measuring the right audience cluster?" — See metrics for See, Do metrics for Do
