## Core Frameworks

### Status Transactions
Every interaction is a status negotiation. High status (dominant, confident) vs low status (submissive, uncertain). Skilled performers can play either. Status shifts create dramatic tension.

### Key Principles
- **"Yes, And"** — Accept and build. Rejection kills narrative momentum.
- **"Be Obvious"** — The first idea is usually the right one. Don't overthink.
- **"Try to Fail"** — Paradoxical permission removes the fear that blocks creativity.
- **Status Play** — Stories require status shifts. A character who stays the same status is boring.
