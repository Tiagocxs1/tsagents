# TS Agents

Ecossistema completo de agentes, skills e comandos para IA.

## Estrutura

| Diretório | Conteúdo |
|-----------|----------|
| squads/  | **396 agentes** organizados em 93 squads temáticos |
| skills/  | **904 skills** (playbooks/instruções) em dezenas de categorias |
| commands/| **171 comandos** slash para Claude Code/opencode |

## Como usar

No painel TS Digitais (https://painel.tsdigitais.com):
1. Vá em **Configurações > Agentes > Repositórios GitHub**
2. Adicione: https://github.com/Tiagocxs1/tsagents
3. Os squads ficam disponíveis no **Team Builder**
